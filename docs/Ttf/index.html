Module Tsdl_ttf.Ttf

SDL2_ttf bindings for use with Tsdl

The UCS-2 Unicode bindings have been omitted; UCS-2 is obsolete, and the current implementation of SDL2_ttf converts such strings to UTF-8 before using them anyway.
